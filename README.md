# 星穹雷电（Raiden-like Shooter）

一个纯前端 Canvas 的纵版飞行射击原型，整体风格参考《雷电3》：

- 玩家战机多段火力、炸弹清屏
- 敌机波次推进（共 30 关）
- 每 5 关精英机强化，并掉落宝石提升火力
- 分数、关卡、生命、宝石等 HUD

## 本地运行

直接在浏览器打开 `index.html` 即可。

如需起本地静态服务器（可选）：

```bash
python3 -m http.server 8080
```

打开 <http://localhost:8080>

## 操作说明

- 移动：`WASD` 或方向键
- 射击：`J` 或空格
- 炸弹：`K`
